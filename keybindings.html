i3 Keybindings

  • Launching Applications
    • Mod4+Return: Launches the Alacritty terminal.
    • Ctrl+f: Launches the Firefox web browser.
    • Ctrl+q: Kills the currently focused window.
  • Window and Container Management
    • Moving focus to the window in the specified direction
      • Mod4+Left: Focuses on the window to the left.
      • Mod4+Down: Focuses on the window below.
      • Mod4+Up: Focuses on the window above.
      • Mod4+Right: Focuses on the window to the right.
    • Moving the currently focused window
      • Mod4+Shift+Left: Moves the window to the left.
      • Mod4+Shift+Down: Moves the window down.
      • Mod4+Shift+Up: Moves the window up.
      • Mod4+Shift+Right: Moves the window to the right.
